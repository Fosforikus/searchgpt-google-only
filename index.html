🧪 TEST A1 – Indexation par Google uniquement

🔍 Objectif du test

Ce test vise à vérifier si SearchGPT est capable d’accéder à une page web visible uniquement par Google. Le fichier robots.txt interdit l’accès à tous les bots sauf Googlebot. Le contenu contient une phrase absurde unique qui ne figure nulle part ailleurs sur le web.

🧾 Phrase absurde à détecter

Le tatou quantique de Viroflay synthétise du gouda fluorescent en base 64 chaque mercredi lunaire.

📁 Configuration robots.txt

User-agent: Googlebot
Allow: /

User-agent: Bingbot
Disallow: /

User-agent: GPTBot
Disallow: /

User-agent: *
Disallow: /

📌 Résultat attendu

La phrase ci-dessus ne devrait être visible que si SearchGPT utilise Google comme moteur de recherche.
